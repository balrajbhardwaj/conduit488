# Public Assets

This folder contains static assets for the Conduit488 website.

## Logo Files

Place the following logo files in this directory:

- `logo.svg` - Main logo (SVG format preferred)
- `logo-white.svg` - White version for dark backgrounds
- `favicon.ico` - Browser favicon

## Usage

In components:
```tsx
import Image from 'next/image'

<Image src="/logo.svg" alt="Conduit488" width={140} height={40} />
```

## Current Logo Description

The Conduit488 logo features:
- Isometric 3D "C" shape
- Glass panels with transparency
- Terracotta inner glow
- Geometric, modular aesthetic

The logo embodies the "Glass Box Philosophy" - showing internal structures while maintaining clean external lines.
